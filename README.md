# HTML_CSS_Project_Two
A modern, fully responsive HTML &amp; CSS landing page template inspired by Kasper. Includes clean layout sections: hero, services, portfolio grid, video banner, pricing tables, testimonials, and contact form. Built using pure HTML5, CSS3, Normalize.css, and Font Awesome. Perfect for practicing front-end skills or using as a base for custom projects.
